# Brazil-Ecommerce-Analysis
# 🛒 巴西电商数据分析项目

## 📌 项目目标
分析巴西某电商平台的销售数据，挖掘用户行为模式，评估物流效率，并为业务优化提供数据支持。

## 🛠️ 技术栈
- **数据清洗与分析**: Python (Pandas, NumPy, Jupyter Notebook)
- **数据库与查询**: SQL
- **可视化**: Tableau
- **版本控制**: Git & GitHub
